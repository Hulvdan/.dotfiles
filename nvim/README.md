# nvim

## Installation

### Good OS

```bash
ln -s $(pwd)/nvim ~/.config/nvim
```

Inside nvim use `:PackerSync` command.

### Windows

Run in powershell with administrator privileges

```
ni C:\Users\user\AppData\Local\nvim -i SymbolicLink -ta "C:\Users\user\dev\.dotfiles\nvim"
```

Inside nvim use `:PackerSync` command.
